# MAS_projekt

## Overview
This project is a full-stack web application built using Java technologies. It was developed as part of an academic project and demonstrates a classic server-side architecture based on Servlets and JSP.

The application allows users to browse and purchase products, while sellers can manage inventory and process orders.

---

## Tech Stack
- Java (Servlets, JSP)
- Apache Tomcat
- Apache Derby (database)
- HTML / CSS

---

## Features

### Client
- Browse available products
- View product details
- Purchase products

### Seller
- Add new products
- View existing orders
- Process and complete orders

---

## Architecture
The application follows a traditional Java web architecture:

- **Servlets** – handle incoming HTTP requests and business logic  
- **JSP (JavaServer Pages)** – generate dynamic views  
- **Apache Derby** – stores application data (products, orders, users)
